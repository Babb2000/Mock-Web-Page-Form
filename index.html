C++
C++ is a programming language written by Barjne Stroustrup in 1979. Written as an extension to C, C++ offered
object-oriented features with data hiding, static binding, and inheritance with low levels of abstraction. C++ makes
programming in the real world more efficient by being able to write directly to low levels of hardware at run-time.
